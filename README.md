# Email Marketing Templates

Coleção de templates de email marketing desenvolvidos em HTML com foco em compatibilidade entre clientes de email como Gmail, Outlook e Apple Mail.

## Templates incluídos

### Email de Boas-vindas
Template utilizado para onboarding de novos usuários.

### Email Promocional
Modelo de campanha promocional para e-commerce.

### Newsletter
Template de newsletter para envio de conteúdo e atualizações.

## Tecnologias utilizadas

- HTML
- Estrutura baseada em tables (email layout)
- CSS inline
- Bulletproof buttons (compatível com Outlook)
- Layout responsivo

## Objetivo

Praticar desenvolvimento de emails profissionais utilizados em campanhas de marketing digital.
